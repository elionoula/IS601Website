History of the Internet

This is a project for the course IS601 Web Systems and Development offered at the New Jersey Institute of Technology, taught by Professor Williams. Throughout this website we will discuss the history of the internet. The internet is defined as a loosely organized international collaboration of autonomous, interconnected networks, which support host to host communication through voluntary adherence to open protocols and procedures.

This website was created by Anik Chattopadhyay and Elio Nicolas.
